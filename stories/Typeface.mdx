# Typefaces

This system design defines **two typefaces**, one serif and one sans-serif.

## Serif

**Font:** Source Serif  
**Weights:** Variables, _100, 200, 300, 400, 500, 600, 700, 800, 900_

<Typeset
  fontSizes={[14, 16, 20, 24, 28, 32, 40]}
  fontWeight={700}
  sampleText={json.sys.typeface.serif}
  fontFamily={json.sys.typeface.serif}
/>

## Sans-serif

**Font:** Open Sans  
**Weights:** Variables, _100, 200, 300, 400, 500, 600, 700, 800, 900_

<Typeset
  fontSizes={[14, 16, 20, 24, 28, 32, 40]}
  fontWeight={700}
  sampleText={json.sys.typeface.sans}

/>

## Usage

The typefaces are defined as variables and can be used within your projects.

### Javascript

```js
import { json } from "@sebastienbarbier/design-system";
// or
import json from "@sebastienbarbier/design-system";
```

You can call `json.sys.typeface` to access the following values:

<div style={{ paddingBottom: "30px" }}>
  <code>
    <pre style={{ fontSize: "0.8em" }}>
      {`
${JSON.stringify(
  {
    sys: {
      typeface: json.sys.typeface,
    },
  },
  null,
  2,
)}
`}
    </pre>
  </code>
</div>

### CSS

Use the `var` function within your CSS/SCSS to access one of the following variables:

```css
--sys-typeface-serif-variable: "Source Serif Variable", "Source Serif", Arial,  serif 
--sys-typeface-sans-variable: "Open Sans Variable", "Open Sans", Helvetica, sans-serif 
--sys-typeface-serif: "Source Serif", Arial, serif 
--sys-typeface-sans: "Open Sans", Helvetica, sans-serif;
```

## Add fonts reference within your projects

### Load local fonts

You can load the design system typefaces by importing the `fonts.css` file.

```scss
@use "@sebastienbarbier/design-system/fonts";
```

This will include web fonts within your repository as self hosted. An alternative solution could be to load them from a CDN.
