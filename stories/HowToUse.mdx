# How to use

This design system is bundle as a npm package and can be used within your project as a javascript object or as a scss module.

To add `@sebastienbarbier/design-system` package to your project, install it using the command:

```zsh
npm install @sebastienbarbier/design-system
```


## Javascript 

The  can be accessed as object within your javascript file

```jsx
import { json } from '@sebastienbarbier/design-system';
```

You can then access as a javascript object which contains the following keys:

<code><pre style={{ fontSize: '0.8em' }}>
{ JSON.stringify(json, null, 2) }
</pre></code>

## Within SCSS

Within you scss project, you can use the following SCSS line to load all variables.
```scss
@use "@sebastienbarbier/design-system/style";

.body {
    background: var(--sys-color-background);
}
```
